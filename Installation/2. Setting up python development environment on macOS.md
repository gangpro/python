* 환경설정 파일 중간쯤? 아래와 같은 내용 수정


        $ (기존)
        ## The directory to use for notebooks and kernels.
        #c.NotebookApp.notebook_dir = ''
    
        $ (수정)
        ## The directory to use for notebooks and kernels.
        c.NotebookApp.notebook_dir = '/Users/kang/Documents/Python/python_ML'
         ↳ c.앞에 # 주석 제거
         ↳ jupyter notebook이 실행할 경로 설정
         ↳ 설정한 경로에 가서 새폴더 생성
