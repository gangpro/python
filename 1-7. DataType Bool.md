# 7. 논리형 (bool)
* 논리형 (bool)에는 True와 False가 있다.
* python은 다음과 같은 경우에 False로 간주한다.

## False keyword
* 숫자 0은 False, 나머지 숫자는 True로 간주
* 빈 문자열 (" ") -> False로 간주
  - 문자열("N") -> True로 간주.
* 빈 list ( [] ) -> False로 간주
  - list([N]) -> True로 간주.
* 빈 tuple ( () ) -> False로 간주
  - tuple((N)) -> True로 간주.
* 빈 dict ( { } ) -> False로 간주
  - dict({N}) -> True로 간주.
* None -> False로 간주
